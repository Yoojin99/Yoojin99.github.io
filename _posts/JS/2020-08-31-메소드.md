함수처럼 생겼는데 앞에 점이 붙여져 있으면 그냥 메소드다.

```js
var obj = {
  a:1,
  b: function bb(){
    console.log(this.a);
  },
  c: function(){
    console.log(this.a);
  }
  
  console.dir(obj.b);
  console.dir(obj.c);
}
```

1. 전역스코프 생성
2. 변수 obj 선언
3. 객체 생성
4. obj.b 메소드 호출
  1. obj.b 실행컨텍스트 생성
  2. this에 obj 바인딩. 메소드는 this를 바인딩한다. .앞에까지가 this임.
